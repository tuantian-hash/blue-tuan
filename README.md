## Nguyên tắc bản quyền của repo này

- **Không copy nguyên code/tài liệu** của người khác trừ khi license (MIT / Apache-2.0 / CC) cho phép, và **luôn giữ attribution + file LICENSE gốc**.
- Với repo license không rõ hoặc chỉ liệt kê (awesome-list), repo này **chỉ dẫn link + tóm tắt**, không sao chép.
- Mỗi thư mục vendor có file `SOURCE.md` ghi rõ nguồn, license, commit/ngày lấy về.

## ⚠️ Lưu ý quan trọng về "vá lỗi liên tục"

Không phải repo nào cũng còn được bảo trì. Xem cột **"Còn bảo trì?"** trong [catalog](docs/catalog.md).
Ví dụ: bộ template PMBOK phổ biến nhất (`PMBOK-doc-templates`) **đã bị archive 03/2026** và chuyển sang Codeberg — vẫn dùng tốt nhưng **không còn nhận vá lỗi trên GitHub**.
