# Drawing Review Guide for Bid Evaluation

Loaded during Step 1b (Drawing Review). Use to identify scope items visible on drawings but not explicit in specs — these differentiate thorough bidders from those who only read specs.

## By Drawing Type

### Floor Plans (A-1.x)
- Room count and names → finish quantities
- Partition types at each wall → drywall scope, rated assemblies
- Door marks → hardware sets, frame types
- Casework/millwork symbols → Div 06/12 scope
- Equipment callouts → MEP coordination
- Floor finish transitions → transition strip scope
- Column grid intersections → structural scope reference

### Reflected Ceiling Plans (A-2.x)
- Ceiling types and heights per room
- Light fixture count and type → electrical scope
- Diffuser/grille count → HVAC scope
- Sprinkler head locations → fire suppression scope
- Speaker/strobe locations → fire alarm/AV scope
- Access panel locations → ceiling sub scope
- Soffits and bulkheads → framing scope

### Elevations (A-3.x, A-5.x)
- Material callouts on exterior → cladding scope
- Window/storefront count and size → glazing scope
- Control/expansion joint locations → sealant scope
- Signage locations → specialty scope
- Louver count → HVAC scope

### Details (A-3.x, A-6.x)
- Specific product callouts (manufacturer, model)
- Assembly layers (substrate, membrane, finish)
- Flashing conditions → sheet metal / roofing scope
- Sealant joint types and locations
- Blocking/backing requirements for mounted items

### Structural (S-series)
- Foundation type → concrete scope
- Steel connection types → fabrication scope
- Embed schedule → cast-in items before pour
- Slab depressions and thickened slabs → concrete scope
- Expansion joint locations → multiple trades affected

### MEP Plans (M/P/E-series)
- Equipment tags and counts → unit pricing verification
- Routing complexity (exposed vs. concealed, tight spaces)
- Fixture counts → plumbing scope
- Panel locations and circuit counts → electrical scope
- Ductwork sizes and routing → sheet metal scope

### Schedules (embedded or A-8.x/A-9.x)
- Door schedule: count by type, hardware sets, fire ratings
- Window schedule: count by type, size, glazing spec
- Finish schedule: materials by room → flooring/painting quantities
- Equipment schedule: size, capacity, utility requirements

## Drawing-Derived Scope Items

Items visible on drawings that specs may not quantify:

| Drawing Feature | Derived Scope |
|----------------|---------------|
| Corner count on plan | Corner guard quantity (Div 10) |
| Linear feet of base on plan | Base trim/cove base quantity (Div 09) |
| Fixture count on plumbing plan | Accessory count (soap, towel, TP — Div 10) |
| Equipment tags on mech plan | Disconnect count (Div 26) |
| Rated partition count | Firestopping scope (Div 07) |
| Slab depression count | Waterproofing/fill scope (Div 03/07) |
| Penetration count through rated walls | Firestop system count |
| Roof penetration count | Curb/flashing count (Div 07) |
| Elevator count | Recall wiring, sump pump, ventilation (Div 14/26/22) |

## Quantity Cross-Checks

Use drawing-derived quantities to sanity-check bid quantities:

- **Fixture count from plumbing plan** vs. bidder's fixture count
- **Door count from schedule** vs. hardware set count in bid
- **Room count from finish schedule** vs. painting/flooring SF
- **Equipment count from mech schedule** vs. electrical disconnect count
- **Rated wall LF from life safety plan** vs. firestopping bid

Significant discrepancies (>10%) suggest the bidder didn't review drawings thoroughly — flag as scope gap risk.
