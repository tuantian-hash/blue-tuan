# Coordination Matrix

When a query touches ANY of these interfaces, actively check both scopes for conflicts.

| Interface | Scope A | Scope B | What Fails |
|---|---|---|---|
| Slab depressions | Concrete | Flooring | Depth doesn't account for full assembly (membrane + mortar + tile) |
| Underslab MEP | Concrete | Plumbing/Elec | Rough-in not coordinated before SOG pour |
| Structural embeds | Concrete | Steel | Anchor bolts missed before pour = post-installed at 10x cost |
| Below-grade WP | Concrete | Waterproofing | Membrane termination not detailed at grade transition |
| Shelf angles | Masonry | Steel | Misaligned with structure; deflection criteria not met |
| Cavity drainage | Masonry | Flashing | Through-wall flashing not continuous; weeps blocked |
| Fireproofing vs MEP | Fireproofing | All MEP | Hangers through SFRM without patching protocol |
| Air barrier at windows | Air Barrier | Windows/CW | Discontinuous at frame = #1 air leakage location |
| Perimeter firesafing | Curtain Wall | Firestopping | Slab edge gap not firestopped at each floor |
| Rated partition penetrations | Drywall | All MEP | MEP through rated wall without listed firestop |
| Blocking for accessories | Carpentry | Specialties | Not installed before drywall closure |
| Ceiling coordination | ACT Ceilings | MEP | Sprinklers, diffusers, lights, speakers not on grid |
| Mech-elec match | HVAC | Electrical | V/Ph/A mismatch between schedules |
| Door HW + access control | Doors/HW | Security | Electric HW not coordinated; power/LV not to frame |
| Fire alarm integration | Fire Alarm | HVAC/Elev/Sprinkler | Missing duct detectors, no recall, unmonitored switches |
| Utility connections | Site Utilities | Building MEP | Pipe/conduit size mismatch at building entry |
| Equipment utilities | Equipment | All MEP | Requirements not provided; rough-in mislocated |
| Furniture power/data | Furnishings | Electrical | Outlet locations don't match furniture layout |
| Roof penetrations | Roofing | All MEP | Not flashed; curbs inadequate; no walkway pads |
| Slab flatness vs flooring | Concrete | Flooring | FF/FL insufficient for specified flooring type |
| Plumbing vs arch fixtures | Plumbing | Architectural | Fixture counts/locations mismatch |
| Kitchen + suppression | Equipment | Fire Suppression | Commercial hood without Ansul/wet chemical |
| Elevator + fire systems | Conveying | Fire Alarm | No Phase I/II recall; no pit sump; no MR ventilation |
| Rated walls + doors | Life Safety | Doors/HW | Rated wall with non-rated door or missing closer/seals |
| Roofing + PV/Solar | Roofing | Electrical/Structural | Penetration flashing, structural loading for array, conduit routing, warranty coordination |
| ICF + embeds + equipment | Concrete | Structural/MEP | Embedded plates and sleeves must be placed before pour; post-install anchors require redesign |
| Warranty: multi-division | Roofing | Flashing/Sealant | Single-source manufacturer requirement — incompatible products void warranty |
| VFD + mech room cooling | HVAC Equipment | Mechanical | VFDs generate significant heat; room cooling must account for full VFD load |
| Controls wiring | Electrical | Building Automation | BACnet/controls wiring routing conflicts with power conductor separation requirements |
