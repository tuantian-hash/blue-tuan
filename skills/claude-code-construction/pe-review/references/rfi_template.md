# PE Expertise — RFI-Ready Output Template

When a conflict, gap, or ambiguity is detected, format the output so a PE can directly draft an RFI:

```
RECOMMENDED RFI:

Subject: [Concise description of issue]

References:
- Drawing: [Sheet number, detail number]
- Specification: [Section number, paragraph]
- Additional: [Any other relevant references]

Issue:
[Clear description of the conflict, gap, or ambiguity]

Suggested Resolution:
[If appropriate, suggest what the PE believes the intent is
based on available information, but clearly label this as the
contractor's interpretation requiring design team confirmation]

Impact:
[Identify affected trades, potential schedule impact,
and whether the issue is blocking procurement or installation]
```
